# Python Port Scanner

A simple TCP Port Scanner built using Python sockets.

## Features

- Scans ports 1-65535
- Resolves hostnames to IP addresses
- Displays open ports
- Handles invalid hostnames and connection errors
- Easy to use from the command line

## Technologies Used

- Python 3
- Socket module
- Datetime module
